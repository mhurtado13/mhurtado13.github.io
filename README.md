# mhurtado13.github.io

Personal portfolio of Marcelo Hurtado — PhD student in Computational Biology and Bioinformatics at the Cancer Research Center of Toulouse (CRCT).

Built with React, TypeScript, Vite and Tailwind CSS, deployed to GitHub Pages via GitHub Actions.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Deployment to GitHub Pages happens automatically on push to `main` (see `.github/workflows/deploy.yml`).
